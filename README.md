# Daily:DO

> 하루 한 번, 작은 실천으로 오늘을 조금 더 다정하게.

[![Image](https://github.com/user-attachments/assets/52a3e47c-3b0e-4d6f-a6e8-ab5875038fb0)](https://www.daily-do.com/)

Daily:DO는 매일 작은 미션을 수행하고 하루를 기록하는 웹 애플리케이션입니다.
성취를 재촉하는 대신, 하루에 작은 결을 남기는 데서 출발했습니다.

할 일은 늘 넘치지만, 하루는 항상 흐릿하게 지나갑니다.
생산성 앱은 많지만, 작은 행동 하나를 **다정하게 기록**해주는 앱은 드뭅니다.
게이미피케이션보다는, 일상에 **잔잔한 결**을 남기고 싶었습니다.

그래서 하루 한 번 작은 미션을 받고, 수행하고, 모으는 루틴을 만들었습니다.

## 📝 주요 기능

### 🎯 오늘의 미션

- 매일 오전 5시, 10개의 미션이 새로 도착합니다.
- 원하는 미션을 최대 5개까지 골라 담을 수 있습니다.
- 미션 완료 시 사진과 메모를 함께 기록할 수 있습니다.

### 🗂️ 마이컬렉션

- 완료한 미션에 따라 컬렉션 획득이 가능합니다.
- 전체 / 획득한 컬렉션 / 미획득 컬렉션으로 탭을 나눠 확인할 수 있습니다.
- 대표 컬렉션을 설정하면 마이페이지에 노출됩니다.

### 📅 마이로그

- 완료한 미션은 월별 캘린더에 기록으로 남습니다.
- 미션 수행 일수, 총 완료 개수, 평균 완료율 등 통계를 제공합니다.
- 날짜를 선택하면 해당 날의 기록을 상세하게 돌아볼 수 있습니다.

### 👤 마이페이지

- 프로필 이미지, 닉네임, 한 줄 소개를 관리할 수 있습니다.
- 관심 카테고리를 설정하면 미션 추천에 반영됩니다.
- 가입일, 최장 연속 달성일, 총 완료 개수 등 활동 통계를 확인할 수 있습니다.
